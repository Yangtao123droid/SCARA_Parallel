1.1、电机的选择  
=

电机因为成本考虑问题和项目工作量问题，没有选择使用伺服电机或者无刷电机作为驱动关节，而是选用的
    
闭环步进电机电机的型号是杰美康的57J1880，在闲鱼上淘的,加驱动器一套大概150元，驱动器的型号是2HSS57，

也是杰美康的闭环驱动器，里面自带了电机控制算法，有电流环位置环等等，控制起来和普通的步进电机驱动器  

一样，发脉冲就好了。
